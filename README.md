# rupaguhaumb-dev.github.io

Personal portfolio site for **Rupa Guha**, deployed via GitHub Pages at
**https://rupaguhaumb-dev.github.io/**.

Single-page hand-written HTML + CSS — no build step, no JavaScript framework.
Edit `index.html` and push to `main` to update the live site.

## Local preview

```
python3 -m http.server 8000
# open http://localhost:8000
```

## Files

- `index.html` — the whole site
- `assets/featured_km.png` — featured-project image (4-group KM plot)
